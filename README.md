# Furniture E-Commerce Landing Page

A sleek, modern, and fully responsive furniture store landing page designed to provide a premium shopping experience. 

## 🚀 Features
* **Interactive Hero Search:** Filter furniture by materials (wood, metal, bamboo), type, and price range.
* **Product Collections:** Dynamic grid layouts showcasing "Best Sale" items with color variants and quick-cart icons.
* **Shop by Room:** Clean navigational categories for Living Room, Bed Room, Dining Room, and more.
* **Media Integrated:** Embeds background video elements highlighting product quality and delivery features.
* **Fully Responsive:** Tailored stylesheets for seamless viewing across mobile, tablet, and desktop devices.

## 🛠️ Built With
* HTML5 (Semantic Structure)
* CSS3 (Custom Layouts & Responsive Media Queries)
* Boxicons & FontAwesome (UI Icons)
